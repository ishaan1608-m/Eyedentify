# Eyedentify
EyeDentify is an AI-powered smart assistive wearable for the visually impaired. It provides real-time object detection, distance estimation, and text reading using OCR with audio feedback. Affordable, hands-free, and user-friendly, it enables safe navigation and independent interaction.
👓 EyeDentify

AI-Powered Smart Assistive Glasses for the Visually Impaired

📌 Overview

EyeDentify is a low-cost, real-time smart wearable designed to empower visually impaired individuals with independence. It combines computer vision, sensors, and audio feedback to help users navigate safely and read printed text effortlessly.

🚀 Features
🔍 Detection Mode
Real-time object detection (person, car, chair, etc.)
Spatial awareness (left, right, center)
Distance estimation using ultrasonic sensor
Low latency, accurate voice feedback
📖 Reading Mode
Captures printed text using camera
OCR extracts text from books/signboards
Text-to-Speech (TTS) reads aloud clearly
🎧 Audio Feedback
Works with Bluetooth earphones/headphones
Hands-free operation
🧠 Tech Stack
Raspberry Pi (3B+ / 5) – Main processing unit
Camera Module – Vision input
Ultrasonic Sensor – Distance measurement
Python – Core programming
OpenCV / AI Models – Object detection
OCR Engine – Text extraction
TTS Engine – Voice output
💡 Key Highlights
💰 Affordable (~₹8000)
⚡ Real-time assistive intelligence
🔄 Dual-mode system (Detection + Reading)
🧩 Smart Fusion (AI + Sensors)
👤 User-friendly & wearable
🎯 Goal

To provide safe navigation and easy access to information, enabling visually impaired individuals to live more independently.

🛠️ Hardware Setup
Raspberry Pi
Camera module
Ultrasonic sensor (HC-SR04)
Bluetooth audio device
Power supply
▶️ How It Works
Camera captures surroundings
AI model detects objects / OCR extracts text
Ultrasonic sensor measures distance
System processes data in real-time
Audio feedback guides the user
